## 🧑‍💻 How to Run 

1. Clone this repository  
   ```bash

    cd CRUD-Starter

2. Frontend Setup
    ```bash
    cd frontend
    npm install
    npm run dev

3. Backend Setup
    ```bash
    cd backend
    npm install
    npm run dev
